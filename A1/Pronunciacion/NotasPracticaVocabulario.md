# Notas de práctica de vocabulario

Este apunte centraliza las referencias de pronunciación que antes dependian de grabaciones. Todas las actividades estan descritas en texto para que puedas prácticarlas solo con el libro o con tu cuaderno.

## Como usar estas notas

- Lee en voz alta cada lista de palabras y marca con subrayado las combinaciones que te resulten difíciles.  
- Escribe al lado una pista fonetica en espanol para recordar el sonido.  
- Alterna lectura silenciosa con lectura dramatizada para fijar ritmo y entonacion sin apoyarte en archivos externos.  
- Repite los ejercicios de las lecciónes (vocales, consonantes, vocabularios tematicos) copiando las palabras que mas se repiten en tu inventario personal.

## Bloques sugeridos

1. **Vocabulario 1-18**: repasa saludos, colores, números y rutinas.  
2. **Listas tematicas**: dias, meses, frutas, interrogativos, emociones.  
3. **Pronunciación y gramática basica**: vocales, combinaciones de consonantes, números compuestos.

Cada bloque se apoya en lecturas y dictados escritos para que el curso siga siendo completamente autosuficiente.
