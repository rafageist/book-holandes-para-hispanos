# Bienvenida y cómo usar este curso

¡Bienvenido/a al curso A1 de Holandés para hispanohablantes!

Este curso está pensado para lectores de nivel básico. Aquí encontrarás lecciones breves, vocabulario temático, guías de pronunciación y ejercicios prácticos.

Cómo usarlo:

1. Sigue las lecciones en orden: `01_Les1` → `02_Les2` → ... → `08_Les8`.
2. Usa la carpeta `Pronunciacion` para revisar las vocales y consonantes cuando aparezcan en las lecciones.
3. Amplía vocabulario con los archivos en `Vocabulario/`.
4. Realiza los ejercicios en `Ejercicios/`.

Todas las notas están en español y pensadas para un lector A1.
