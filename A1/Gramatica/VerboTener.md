# Gramática: El verbo "hebben" (tener)

El verbo "hebben" se usa para expresar posesión.

Conjugación presente:
- ik heb (yo tengo)
- jij hebt (tú tienes)
- hij/zij/het heeft (él/ella/eso tiene)
- wij hebben (nosotros tenemos)
- jullie hebben (vosotros tenéis)
- zij hebben (ellos tienen)
