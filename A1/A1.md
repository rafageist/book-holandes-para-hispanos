# Índice maestro — Curso A1

Esta nota es el índice principal del curso A1. Desde aquí puedes navegar por los módulos y las notas índice de cada carpeta.

## Introductie / Bienvenida

- Guía de acceso al curso y descarga de libros (paso a paso).
- Explicación del sistema de tareas y cómo organizar el vocabulario personal.
- Recomendaciones para configurar aplicaciones de estudio en cualquier dispositivo.
- Listado de recursos impresos/externos: playlists, podcasts, artículos.
- Contexto cultural: Países Bajos vs Holanda, colonias, himno y bandera.
- Resumen de saludos y fórmulas de cortesía (*Buenos días*, *Hoe gaat het?*, *Alsjeblieft / Dank je wel*, uso de **graag**).
- Checklist de hábitos diarios para estudiar sin depender de recursos externos.
