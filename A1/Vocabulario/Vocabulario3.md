# Vocabulario 3 - Profesiones y lugares de trabajo

Este bloque acompaña las lecciones de familia, planificación y metas profesionales. Incluye profesiones comunes, lugares y expresiones útiles para hablar de tareas laborales o académicas.

## Profesiones y cargos

| Holandés | Español | Nota |
|----------|---------|------|
| `de docent / de leraar` | docente | `docent` es más formal (educación superior). |
| `de student` | estudiante | Masculino y femenino. |
| `de verpleegkundige` | enfermero/a | Pronuncia `vër-pleijs-kún-de-gue`. |
| `de arts` / `de dokter` | médico | `arts` se usa en contextos formales. |
| `de verkoper` / `de verkoopster` | vendedor/a | Termina en `-er` / `-ster`. |
| `de bakker` | panadero | Consonante doble para mantener la vocal corta. |
| `de monteur` | mecánico | Diptongo `eu`. |
| `de chauffeur` | chofer | Origen francés, suena “shó-fœr”. |
| `de programmeur` | programador | Útil en contextos modernos. |
| `de manager` | gerente | Anglicismo integrado. |

## Lugares de trabajo y estudio

| Holandés | Español | Ejemplo |
|----------|---------|---------|
| `het kantoor` | oficina | `Ik werk op kantoor.` |
| `de winkel` | tienda | `Mijn zus werkt in een winkel.` |
| `het ziekenhuis` | hospital | `Hij werkt in het ziekenhuis.` |
| `de fabriek` | fábrica | `Zij werken in een fabriek.` |
| `de bouwplaats` | obra de construcción | `De monteur is op de bouwplaats.` |
| `de school / de universiteit` | escuela / universidad | `Wij studeren aan de universiteit.` |
| `het restaurant` | restaurante | `De kok werkt in een restaurant.` |
| `het thuiswerk` | trabajo desde casa | `Ik doe één dag per week thuiswerk.` |

## Herramientas y tareas

| Holandés | Español | Comentario |
|----------|---------|------------|
| `de computer` | computadora | `Ik werk achter de computer.` |
| `de telefoon` | teléfono | `Bel je de klant?` |
| `de meeting / de vergadering` | reunión | `Wij hebben een vergadering om negen uur.` |
| `het rapport` | informe | `Ik schrijf een rapport.` |
| `de deadline` | fecha límite | Anglicismo común. |
| `het contract` | contrato | `Ik teken het contract.` |
| `de ploegendienst` | turno | Útil para describir horarios. |

## Frases útiles

- `Ik werk fulltime / parttime.`
- `Zij heeft een nieuwe baan als verpleegkundige.`
- `Wij volgen een cursus Nederlands op het werk.`
- `Hij zoekt een flexibele job in de IT-sector.`
- `Mijn collega schrijft elke week een rapport.`

## Mini práctica

1. Completa las frases: `Mijn broer is ____ (profesión) en hij werkt in ____ (lugar).`
2. Escribe tres oraciones describiendo tus tareas diarias (`Ik bel klanten`, `Ik maak afspraak`).
3. Traduce al neerlandés: “Ella estudia enfermería en la universidad y trabaja medio tiempo en una tienda”.
4. Diseña una tarjeta de presentación simple (Nombre, profesión, empresa, teléfono) usando vocabulario neerlandés.

> Consejo: Mezcla estas palabras con los ejercicios de las lecciones 5 y 6 para practicar verbos en presente y ordenar la oración cuando el tiempo va al inicio.
