# Vocabulario 2 - Rutina y espacios cotidianos

Este bloque amplía el repertorio básico que aparece en las primeras lecciones. Practica objetos del hogar, verbos frecuentes y expresiones útiles para describir tu día.

## Objetos y habitaciones

| Holandés | Español | Nota |
|----------|---------|------|
| `de woonkamer` | sala | Recuerda que `woon-` se relaciona con vivir. |
| `de slaapkamer` | dormitorio | Usa `ik slaap` (yo duermo). |
| `de badkamer` | baño | Incluye ducha y lavabo. |
| `de keuken` | cocina | Verbo útil: `koken` (cocinar). |
| `de tafel` | mesa | Singular `de tafel`, plural `de tafels`. |
| `de stoel` | silla | `Ik zit op de stoel` = estoy sentado. |
| `het raam` | ventana | Observa el diptongo AA. |
| `de deur` | puerta | U larga (`de-ur`). |
| `de kast` | armario | Muy usada en inventarios de casa. |
| `de sleutel` | llave | Útil para historias cotidianas. |

## Verbos y expresiones de rutina

| Verbo / Expresión | Significado | Ejemplo |
|-------------------|-------------|---------|
| `opstaan` | levantarse | `Ik sta om zeven uur op.` |
| `ontbijten` | desayunar | `Wij ontbijten samen.` |
| `werken` / `studeren` | trabajar / estudiar | `Hij werkt thuis`, `Ik studeer Nederlands.` |
| `boodschappen doen` | hacer la compra | `Ik doe boodschappen op zaterdag.` |
| `koken` | cocinar | `Zij kookt pasta.` |
| `schoonmaken` | limpiar | `Wij maken de keuken schoon.` |
| `uitrusten` | descansar | `Ik rust uit op de bank.` |
| `televisie kijken` | ver televisión | `Kijk je tv om acht uur?` |

## Frases modelo

- `Ik werk 's ochtends in de woonkamer.`
- `Mijn dochter studeert in haar slaapkamer.`
- `De sleutel ligt op de tafel.`
- `Wij maken elke maandag de badkamer schoon.`
- `Na het werk rust ik uit met een boek.`

Uso recomendado: reemplaza el sustantivo resaltado y vuelve a leer la frase en voz alta. Así interiorizas la pronunciación y el orden básico (sujeto + verbo + complemento).

## Mini práctica escrita

1. Describe tu casa en cuatro frases. Incluye al menos dos habitaciones y un objeto con su ubicación (`op`, `naast`, `achter`).
2. Completa las frases con un verbo del cuadro:
   - `Ik ____ om zes uur op.`
   - `Wij ____ de woonkamer schoon.`
   - `Zij ____ boodschappen in de supermarkt.`
3. Escribe una mini agenda de lunes a viernes indicando una acción por día (`Maandag: Ik werk thuis.`).
4. Traduce al neerlandés: “Después de cenar descanso en el sofá”.

> Consejo: combina este vocabulario con los ejercicios de la lección 5 para reforzar la planificación del hogar y la familia.
