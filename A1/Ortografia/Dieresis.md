# Dieresis / Trema

El `trema` (dos puntos sobre la vocal, por ejemplo `ë`) se usa cuando dos vocales consecutivas deben pronunciarse por separado. Reglas clave:

1. **Palabras con vocal+vocal**: `reünie`, `patiënt`.
2. **Plurales y sufijos**: `idee` → `ideeën`, `knie` → `knieën`.
3. **Números compuestos**: `tweeëntwintig`, `drieënveertig` (la unidad termina en vocal y la decena comienza con otra vocal).
4. **Nombres geográficos**: `België`, `Curaçao`, `Haïti`.
5. **No se usa** cuando el origen extranjero ya sugiere la pronunciación (`opticien`, `petrolie`) o cuando las palabras se mantienen separadas por un guion (`vanille-ijs`).

Práctica sugerida:

- Reescribe tu lista de palabras difíciles marcando las que necesitan trema.
- Crea 5 números compuestos (20-100) y añade el signo donde corresponda.
