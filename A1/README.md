# Curso A1 — Estructura

Esta carpeta contiene el curso A1 reorganizado para lectores hispanohablantes (nivel básico).

Estructura propuesta:

- `00_Introduccion/` — Bienvenida y cómo usar este curso.
- `01_Les1/` … `08_Les8/` — Lecciones 1 a 8 (cada carpeta incluye `Indice.md` y `Leccion.md`).
- `Vocabulario/` — Listas temáticas: colores, familia, números, alimentos, etc.
- `Pronunciacion/` — Guías de vocales, consonantes, ejemplos y plantillas.
- `Gramatica/` — Artículos, verbos básicos, estructura de oraciones.
- `Ejercicios/` — Ejercicios y plantillas para practicar.
- `RecursosVisuales/` — Imágenes, diagramas y plantillas.

Comunicación: todas las notas están en español y dirigidas a un lector de nivel A1.
