# Vocabulario - Leccion 8

## Pronombres posesivos

| Persona | Singular | Nota |
|---------|----------|------|
| ik | `mijn` | |
| jij | `jouw / je` | informal |
| hij | `zijn` | |
| zij | `haar` | |
| wij | `ons` (het-woorden) / `onze` (de-woorden) | `ons huis`, `onze familie` |
| jullie | `jullie` | |
| zij (plural) | `hun` | |

Frases: `Dit is mijn boek`, `Onze docent is erg vriendelijk`.

## Pronombres objeto (complementarios)

`mij/me`, `jou/je`, `hem`, `haar`, `ons`, `jullie`, `hen/hun/ze`.  
Ejemplos:
- `Kun je mij helpen?`
- `Ik zie hem morgen.`
- `Zij geeft hun het cadeau.`

## Números 100+

| Numero | Holandes |
|--------|----------|
| 100 | `honderd` |
| 125 | `honderd vijfentwintig` |
| 342 | `driehonderd tweeënveertig` |
| 999 | `negenhonderd negenennegentig` |
| 1.000 | `duizend` |
| 1.500 | `vijftienhonderd` o `duizend vijfhonderd` |

Frases: `Het kost tweehonderd euro`, `Er wonen duizend mensen in dit dorp`.

## Vocabulario afectivo (“Palabras de amor”)

`liefde`, `woorden`, `hart`, `herinnering`, `verlangen`, `ik hou van jou`, `voor altijd`.  
Úsalos para escribir un mensaje de cierre: `Ik stuur je woorden van liefde`, `Ik hou van jou`.

## Expresiones de cierre

- `Bedankt voor alles`, `Tot de volgende les`, `Veel succes met A2`.
- `Ik ben trots op...`, `Ik wens je veel geluk`.
