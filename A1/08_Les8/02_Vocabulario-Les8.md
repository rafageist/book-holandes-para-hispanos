# Vocabulario - Les8

## Pronombres posesivos

| Persona | Posesivo | Ejemplo |
|---------|----------|---------|
| ik | `mijn` | `Dit is mijn boek.` |
| jij/je | `jouw/je` | `Wat is jouw plan?` |
| hij | `zijn` | `Zijn kamer is groot.` |
| zij/ze | `haar` | `Haar fiets is nieuw.` |
| wij | `ons/onze` | `Ons huis is klein`, `Onze familie is hier.` |
| jullie | `jullie` | `Jullie idee is goed.` |
| zij (plural) | `hun` | `Hun kinderen studeren.` |

Regla: usa `ons` para sustantivos neutros singulares, `onze` para los demas.

## Pronombres objeto

`mij/me`, `jou/je`, `hem`, `haar`, `ons`, `jullie`, `hen/hun/ze`.  
Ejemplos:
- `Kun je mij helpen?`
- `Ik zie hem morgen.`
- `Wij bellen jullie later.`

## Números 100+

| Número | Neerlandes |
|--------|------------|
| 100 | `honderd` |
| 125 | `honderd vijfentwintig` |
| 342 | `driehonderd tweeenveertig` |
| 999 | `negenhonderd negenennegentig` |
| 1.000 | `duizend` |
| 1.500 | `vijftienhonderd` / `duizend vijfhonderd` |

Frases: `Het kost tweehonderd euro`, `Er wonen duizend mensen in dit dorp`.

## Expresiones de cierre

`Bedankt voor alles`, `Tot de volgende les`, `Veel succes met A2`, `Ik ben trots op jou`, `Ik wens je veel geluk`, `Ik stuur je woorden van liefde`.

## Vocabulario afectivo (lectura)

`liefde`, `woorden`, `licht`, `warmte`, `herinnering`, `verlangen`, `ik hou van jou`, `ik geloof in jou`.  
Usa estas palabras para escribir mensajes motivacionales.

## Planificacion A2

Verbros utiles para metas: `plannen`, `voorbereiden`, `volhouden`, `oefenen`, `opschrijven`, `evalueren`.  
Ejemplos: `Ik plan mijn studie`, `Ik blijf oefenen`, `Ik schrijf mijn doelen op`.
