## Panorama

En esta primera lección del nivel A2 nos enfocamos en narrar experiencias recientes y pasadas. Compararemos el **perfecto compuesto** (`ik heb gewoond`) con el **pasado simple** (`ik woonde`) y prácticaremos cómo escribir un diario personal o un relato corto. También repasaremos verbos irregulares frecuentes y conectores temporales.

## Objetivos

- Diferenciar cuándo usar pasado simple vs perfecto.
- Narrar eventos cotidianos con conectores (`toen`, `terwijl`, `daarna`, `uiteindelijk`).
- Ampliar el inventario de verbos irregulares más usados en relatos personales (gaan, komen, zien, doen, denken, schrijven, eten, drinken, blijven, beginnen).
- Escribir un mini diario (5-6 frases) mezclando tiempos verbales.

## 1. Perfecto compuesto vs pasado simple

| Uso | Perfecto (`hebben/zijn + participio`) | Pasado simple |
|-----|---------------------------------------|---------------|
| Acción puntual con efecto presente | `Ik heb in Utrecht gewoond (en nu niet meer / relevancia actual).` | `Ik woonde in Utrecht (relato histórico, sin conexión actual).` |
| Experiencia reciente | `Wij hebben gisteren veel gewandeld.` | `Wij wandelden gisteren veel.` |
| Historias largas (literatura, noticias antiguas) | Menos frecuente | Preferido (`Hij werkte twintig jaar bij Philips.`) |

Regla práctica:
1. ¿Importa el resultado actual? → usa perfecto.
2. ¿Es una historia cronológica, sin relación al presente? → usa pasado simple.

### Ejemplo narrativo

```
Gisteren heb ik mijn oude vrienden bezocht. We hebben koffie gedronken en veel gelachen.
Toen ik naar huis fietste, begon het plotseling te regenen. Ik voelde me toch blij,
want ik had een fantastische dag.
```

## 2. Verbos irregulares clave (extracto)

| Infinitivo | Pasado simple | Participio |
|------------|---------------|------------|
| gaan | ging/gingen | gegaan (met zijn) |
| komen | kwam/kwamen | gekomen (met zijn) |
| zien | zag/zagen | gezien |
| doen | deed/deden | gedaan |
| denken | dacht/dachten | gedacht |
| schrijven | schreef/schreven | geschreven |
| eten | at/aten | gegeten |
| drinken | dronk/dronken | gedronken |
| blijven | bleef/bleven | gebleven (met zijn) |
| beginnen | begon/begonnen | begonnen (met zijn) |

Práctica recomendada:
- Crea tarjetas NL→ES con las tres columnas.
- Escribe frases en ambos tiempos (`Ik deed`, `Ik heb gedaan`) para notar el contraste.

## 3. Conectores temporales

- `toen` (cuando - pasado). `Toen ik klein was, woonde ik in Aruba.`
- `terwijl` (mientras). `Terwijl ik kookte, belde mijn moeder.`
- `daarna`, `daarop`, `later`, `uiteindelijk`.
- `intussen/inmiddels` (mientras tanto / ya).

Actividad: redacta una historia en 5 frases usando al menos 3 conectores diferentes.

## 4. Diario personal – plantilla

1. **Inleiding**: `Vorige week heb ik besloten...`
2. **Gebeurtenis 1** (perfecto) + sentimiento.
3. **Gebeurtenis 2** (pasado simple) + detalle descriptivo.
4. **Reflectie**: `Ik ben blij/teleurgesteld omdat...`
5. **Plan**: `Volgende week wil ik...`

Ejemplo rápido:

```
Vorige maand heb ik een nieuwe baan gevonden.
Elke ochtend stond ik om zes uur op en oefende ik Nederlands in de trein.
Tijdens mijn eerste dag maakte ik kennis met mijn nieuwe team.
Ik voelde me zenuwachtig, maar ik heb veel geleerd.
Volgende week wil ik mijn eerste presentatie geven.
```

## Checklist

1. Repasar la lista de verbos irregulares y completar el cuadro de posesión.  
2. Escribir un diario de 5 frases mezclando perfecto y pasado simple.  
3. Preparar una breve presentación oral (1 minuto) sobre una experiencia pasada usando al menos 3 conectores.  
4. Añadir las palabras nuevas a tu inventario de vocabulario.
