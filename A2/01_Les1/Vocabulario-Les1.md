# Vocabulario – A2 Les1 (Historias personales)

## Verbos irregulares frecuentes

| Infinitivo | Pasado | Participio |
|------------|--------|------------|
| gaan | ging/gingen | gegaan (zijn) |
| komen | kwam/kwamen | gekomen (zijn) |
| blijven | bleef/bleven | gebleven (zijn) |
| beginnen | begon/begonnen | begonnen (zijn) |
| doen | deed/deden | gedaan |
| zien | zag/zagen | gezien |
| schrijven | schreef/schreven | geschreven |
| eten | at/aten | gegeten |
| drinken | dronk/dronken | gedronken |
| denken | dacht/dachten | gedacht |

## Conectores temporales

`eerst`, `toen`, `terwijl`, `daarna`, `vervolgens`, `uiteindelijk`, `intussen`, `inmiddels`.

## Expresiones útiles para diarios

- `Vorige week heb ik ...`
- `Toen ik klein was ...`
- `Ik voelde me ...`
- `Ik was van plan ...`
- `Aan het einde van de dag ...`

## Adjetivos para describir emociones

`nerveus`, `opgelucht`, `enthousiast`, `teleurgesteld`, `trots`, `vermoeid`.

Úsalos con pasado/perfecto: `Ik voelde me nerveus`, `Ik ben trots op mijn werk`.
